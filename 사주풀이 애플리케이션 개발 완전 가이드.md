# 사주풀이 애플리케이션 개발 완전 가이드

## 개요

사주풀이 앱은 전통 명리학(사주팔자)을 기반으로 사용자의 생년월일시를 입력받아 운세를 자동 계산·해석해주는 서비스다. 최근 5년 사이 관련 앱 시장이 약 3배 이상 성장했으며, AI 기술과 결합한 개인 맞춤형 운세 서비스로 진화하고 있다. 이 가이드는 앱 개발에 필요한 명리학적 이론, 계산 알고리즘, 기능 구성, 기술 스택까지 종합적으로 정리한다.[^1][^2]

***

## 1. 사주명리학 핵심 이론

### 1.1 사주팔자(四柱八字)란

사주(四柱)는 '네 개의 기둥'(연주·월주·일주·시주), 팔자(八字)는 각 기둥의 천간·지지 2글자씩 총 '여덟 글자'를 의미한다. 이 8글자가 개인의 운명·성격·인간관계·직업운 등을 담는 코드로 해석된다.[^3]

### 1.2 천간(天干) — 10개

천간은 하늘의 기운, 즉 정신과 이상을 상징하는 10개 요소다.[^4][^5]

| 번호 | 천간 | 한자 | 오행 | 음양 |
|------|------|------|------|------|
| 1 | 갑 | 甲 | 목(木) | 양(+) |
| 2 | 을 | 乙 | 목(木) | 음(-) |
| 3 | 병 | 丙 | 화(火) | 양(+) |
| 4 | 정 | 丁 | 화(Fire) | 음(-) |
| 5 | 무 | 戊 | 토(土) | 양(+) |
| 6 | 기 | 己 | 토(土) | 음(-) |
| 7 | 경 | 庚 | 금(金) | 양(+) |
| 8 | 신 | 辛 | 금(金) | 음(-) |
| 9 | 임 | 壬 | 수(水) | 양(+) |
| 10 | 계 | 癸 | 수(Water) | 음(-) |

### 1.3 지지(地支) — 12개

지지는 땅의 기운, 현실과 환경을 상징하는 12개 요소다. 각 지지는 지장간(支藏干)이라는 내부 천간을 품고 있으며, 여기(餘氣)·중기(中氣)·정기(正氣)로 구성된다.[^5][^6][^3]

| 번호 | 지지 | 한자 | 오행 | 음양 | 시간대(표준) | 시간대(한국 보정) |
|------|------|------|------|------|------------|---------------|
| 1 | 자 | 子 | 수 | 양 | 23:00~01:00 | 23:30~01:30 |
| 2 | 축 | 丑 | 토 | 음 | 01:00~03:00 | 01:30~03:30 |
| 3 | 인 | 寅 | 목 | 양 | 03:00~05:00 | 03:30~05:30 |
| 4 | 묘 | 卯 | 목 | 음 | 05:00~07:00 | 05:30~07:30 |
| 5 | 진 | 辰 | 토 | 양 | 07:00~09:00 | 07:30~09:30 |
| 6 | 사 | 巳 | 화 | 음 | 09:00~11:00 | 09:30~11:30 |
| 7 | 오 | 午 | 화 | 양 | 11:00~13:00 | 11:30~13:30 |
| 8 | 미 | 未 | 토 | 음 | 13:00~15:00 | 13:30~15:30 |
| 9 | 신 | 申 | 금 | 양 | 15:00~17:00 | 15:30~17:30 |
| 10 | 유 | 酉 | 금 | 음 | 17:00~19:00 | 17:30~19:30 |
| 11 | 술 | 戌 | 토 | 양 | 19:00~21:00 | 19:30~21:30 |

### 1.4 육십갑자(六十甲子)

천간 10개와 지지 12개가 음양이 같은 것끼리만 결합(양+양, 음+음)하므로 5×6+5×6=60개의 조합이 만들어진다. 이 60개의 간지 조합이 60년을 한 주기로 순환한다.[^3]

### 1.5 오행(五行)과 상생·상극

오행은 목(木)·화(火)·토(土)·금(金)·수(水)로, 사주풀이의 기본 에너지 체계다.[^8]

- **상생(相生)**: 목→화→토→금→수→목 순서로 서로 생해줌[^8]
- **상극(相剋)**: 목→토, 토→수, 수→화, 화→금, 금→목 순서로 서로 극함[^8]

오행 상생·상극은 십성(육친) 계산의 기본 로직으로 사용된다.[^8]

## 2. 사주팔자 계산 알고리즘

### 2.1 연주(年柱) 계산

태어난 해의 육십갑자로 연주를 결정한다. **주의사항**: 명리학에서 한 해의 시작은 설날이 아니라 입춘(立春)이므로, 1~2월생의 경우 해당 연도의 입춘 절입시각(節入時刻) 이전에 태어났다면 전년도 간지를 적용해야 한다.[^3]

### 2.2 월주(月柱) 계산 — 월건기법(月建起法)

연간(年干)에 따라 아래 표로 월주를 결정한다. 각 월의 시작 기준은 1일이 아니라 해당 절기의 절입시각이다.[^3]

### 2.3 일주(日柱) 계산

일주는 규칙적인 패턴이 없어 만세력 데이터를 직접 참조해야 한다. 실제 구현 시에는 기준일(예: 1900년 1월 1일의 간지)을 알고 날짜 수 mod 60을 이용해 60갑자 인덱스를 계산하는 방식을 쓴다.[^3]

### 2.4 시주(時柱) 계산 — 시기법(時起法)

일간(日干)과 출생 시간으로 시주를 결정한다. **한국 보정값(-30분) 적용 후 시지를 결정**한다.[^7]

|---------|---------|---------|---------|---------|---------|---------|---------|---------|---------|---------|---------|---------|
| 甲/己 | 甲子 | 乙丑 | 丙寅 | 丁卯 | 戊辰 | 己巳 | 庚午 | 辛未 | 壬申 | 癸酉 | 甲戌 | 乙亥 |
| 乙/庚 | 丙子 | 丁丑 | 戊寅 | 己卯 | 庚辰 | 辛巳 | 壬午 | 癸未 | 甲申 | 乙酉 | 丙戌 | 丁亥 |

시간 계산 로직(수학적 패턴):[^7]
- 일간 인덱스(갑=1, 을=2, ..., 계=10)에서 5 초과 시 `-5` 처리
- 시간 천간 인덱스 = `(시지 인덱스 + (일간 보정값 - 1) × 2) mod 10`

### 2.5 대운(大運) 계산 — 10년 주기

대운은 10년 단위로 변하는 운의 흐름으로, 월주를 기준으로 순행 또는 역행하며 산출한다.[^9][^3]

**계산 절차**:
1. 연간의 음양과 성별로 순행/역행을 결정한다
   - 양년(甲·丙·戊·庚·壬) + 남자, 음년(乙·丁·己·辛·癸) + 여자 → **순행**
   - 음년 + 남자, 양년 + 여자 → **역행**[^3]
2. 순행: 출생일~다음 절기 절입시각까지 날짜 수 ÷ 3 = 첫 대운 시작 나이[^10]
3. 역행: 출생일~이전 절기 절입시각까지 날짜 수 ÷ 3 = 첫 대운 시작 나이[^9]
4. 월주부터 순/역행하여 10개 대운 간지를 나열한다

**예시**: 기년(己年, 음년) 2000년 1월 1일 남자 → 역행, 대운수 8세, 乙亥→甲戌→癸酉... 순서[^3]

### 2.6 세운(歲運) 계산

세운은 1년 단위로 변하는 운으로, 해당 연도의 연주(年柱) 간지가 세운의 간지가 된다. 별도 계산 없이 해당 연도의 60갑자 데이터를 참조하면 된다.[^3]

### 3.2 신강·신약(身强·身弱) 판단

일간의 세력 강도를 판단하는 개념으로, 용신(用神) 결정의 전제 조건이다.[^13]

- **신강(身强)**: 비겁 + 인성의 점수 합계 ≥ 50점 (전체 100점 기준)[^13]
- **신약(身弱)**: 식상 + 재성 + 관성의 점수 합계가 더 많음

각 궁위 점수 배분:[^13]
- 월지: 25점, 일지: 15점
- 나머지 6개 궁위(연간·연지·월간·일간·시간·시지): 각 10점

### 3.3 용신(用神)

사주의 중화(中和)를 맞춰주는 핵심 오행으로, 개운법과 진로 결정의 기준이 된다. 주요 용신 도출 방법:[^3]

| 방법 | 설명 |
|------|------|
| **억부법(抑扶法)** | 신강하면 억제(식상·재·관), 신약하면 부조(비겁·인성)하는 오행을 용신으로 삼음 |
| **조후법(調候法)** | 차가운 사주(수·금 과다)엔 목·화, 더운 사주엔 수·금으로 온도 균형 맞춤 |
| **격국법(格局法)** | 월지의 지장간 투출 여부로 격국을 잡고, 격국에 필요한 오행을 용신으로 결정 |
| **통관법(通官法)** | 상극 두 오행이 팽팽할 때 중간 오행으로 통관 |
| **전왕법(全旺法)** | 한 오행이 압도적으로 강할 때 그 오행이 용신 |

### 3.4 지지의 합충형파해(合沖刑破害)

사주 8글자 간의 상호 작용으로, 운세 변화의 핵심 메커니즘이다.[^3]

| 작용 | 종류 | 핵심 의미 |
|------|------|---------|
| **합(合)** | 천간합(5쌍), 지지 육합(6쌍), 삼합(4종), 방합(4종) | 두 기운이 합쳐져 새 오행으로 변화 또는 합거 |
| **충(沖)** | 천간충(4쌍), 지지 육충(6쌍) | 두 기운이 충돌·변동·파괴 |
| **형(刑)** | 삼형·자형·무례지형 | 형벌·사고·수술 등 강제적 변화 |
| **파(破)** | 육파(6쌍) | 진행 중 일의 차질·정체 |
| **해(害)** | 육해(6쌍) | 인간관계의 갈등·원망 |

### 3.5 신살(神殺)

사주에 특정 글자 조합이 있을 때 부여하는 길신(吉神)·흉신(凶神) 개념으로, 대표적인 것으로는 천을귀인(天乙貴人)·역마살(驛馬殺)·도화살(桃花殺)·백호살(白虎殺) 등이 있다.[^14][^3]

### 5.1 만세력 계산 라이브러리

사주 계산의 핵심인 만세력(24절기 기반 날짜 변환)은 직접 구현하거나 공개 라이브러리를 활용할 수 있다.

| 라이브러리 | 언어 | 기능 | 링크 |

| **sajupy** | Python | 1900~2100년 만세력, 음력변환[^20] | github.com/0ssw1/sajupy |
| **laravel-fortune** | PHP/Laravel | 만세력·십신·길신·대운·세운[^21] | github.com/wangta69/laravel-fortune |

### 5.2 기술 스택 권장 구성

**프론트엔드**
- React Native 또는 Flutter — iOS/Android 동시 개발
- Next.js — 웹 버전 및 SEO 최적화
- Tailwind CSS — UI 스타일링

**백엔드**
- Node.js(Express) 또는 Python(FastAPI) — API 서버
- 만세력 계산 모듈 통합
- PostgreSQL / Supabase — 사용자 데이터·명반 저장

**AI 연동**

- 사주 8글자 + 대운·세운 정보를 프롬프트로 전달해 맞춤 해석 생성
- 초기 GPT-3.5-turbo는 사주풀이 품질 낮으므로 GPT-4 계열 사용 권장[^18]

**인프라**
- Vercel / Netlify — 웹 배포
- AWS Lambda — 서버리스 만세력 계산 API

## 결론

사주풀이 앱 개발의 핵심은 **정확한 만세력 계산 엔진**과 **풍부한 해석 콘텐츠**의 결합이다. 만세력 계산(한국 표준시 30분 보정, 절기 기반 절입시각 처리, 야자시 옵션)만 정확히 구현해도 기존 앱과의 신뢰도 차별화가 가능하다. AI API를 활용하면 개인 맞춤형 텍스트 풀이를 빠르게 구현할 수 있으나, GPT 모델 품질 선택(GPT-4 계열 권장)과 프롬프트 설계가 풀이 품질을 좌우한다. 시장 포화에도 불구하고 웹툰 결합·글로벌화·특정 주제 특화(직업궁합, MBTI 연계 등) 등의 차별화 전략으로 신규 진입 여지는 충분하다.[^16][^23][^18][^7]
